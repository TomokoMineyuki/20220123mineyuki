# TODOアプリ

## 概要

Laravelで作成したTODOアプリ  
<http://pacific-bastion-59146.herokuapp.com/>  

## 機能一覧

- 新規タスクの追加  
  入力フォームにタスクを記入し「追加」ボタンを押す  
  下部のリストに追加順にタスクが表示される  
- タスクの更新  
  タスクリスト内の情報を変更した上で「更新」ボタンを押す  
  リストが更新される  
- タスクの削除  
  タスクリストの「削除」ボタンを押す  
  該当のタスクが削除される
